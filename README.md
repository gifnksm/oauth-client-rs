OAuth client for Rust

[Documentation](https://docs.rs/oauth-client/)

## How to use?

Add this to your `Cargo.toml`:

```toml
[dependencies]
oauth-client = "0.3"
```

and this to your crate root:

```rust
extern crate oauth_client;
```

See [examples](./examples).
